1) Instrukcja wyboru – składnia:
if (warunek) { instrukcje }

2) Instrukcja wyboru – schemat:
Warunek → jeśli prawda → wykonaj blok

3) Instrukcja alternatywy – składnia:
if (warunek) { instrukcje } else { inne instrukcje }

4) Instrukcja alternatywy – schemat:
Warunek → TAK/NIE → dwa różne bloki

5) Operatory porównania:
==, !=, >, <, >=, <=

6) Uwaga:
== porównuje wartości, === porównuje też typ

7) Operatory logiczne:
&& (i), || (lub), ! (negacja)
