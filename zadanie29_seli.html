ZADANIE 29 - OBIEKTY W JAVASCRIPT

PYTANIA

1. W jaki sposób można określić właściwość danego obiektu (dwa sposoby wraz z przykładami)?

2. W jaki sposób można odwołać się do pól (właściwości) i metod?

3. Wymień cechy charakterystyczne konstruktora.

4. Konstruowanie obiektu – obiekt ma trzy pola oraz jedną metodę.

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

6. Konstruktor w JS na podstawie przykładu wraz z opisem.

7. Znaczenie słowa kluczowego new wraz z przykładem.

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem.

ODPOWIEDZI

1. Określanie właściwości obiektu

Sposób 1 – zapis z kropką

auto.marka = "Opel Omega";

auto.rok = 1996;

Sposób 2 – zapis z nawiasami

auto["cena"] = 25000;

2. Odwołanie do pól i metod

Do właściwości i metod odwołujemy się przez nazwę obiektu i kropkę.

console.log(samochod1.kolor);

samochod1.jedz();

3. Cechy konstruktora

• Uruchamiany automatycznie przy tworzeniu obiektu
• Ma taką samą nazwę jak klasa
• Nie zwraca wartości
• Może przyjmować argumenty
• Używany razem z new

4. Przykład obiektu (3 pola i 1 metoda)

var osoba_1 = {
nazwisko: "Kowalski",
imie: "Jan",
zawod: "piekarz",
wyswietl: function() {
document.write(this.nazwisko + " " + this.imie);
}
}

5. Słowo this

This oznacza aktualny obiekt. Pozwala odwołać się do jego pól i metod z jego wnętrza.
Pola i metody oddzielamy przecinkami.

6. Konstruktor w JavaScript

function klient(nazwisko, imie, zawod) {
this.nazwisko = nazwisko;
this.imie = imie;
this.zawod = zawod;
this.wypisz = function() {
alert(this.nazwisko + " " + this.imie);
}
}

Konstruktor tworzy obiekty i nadaje im właściwości przy pomocy słowa this.

7. Słowo kluczowe new

New służy do tworzenia nowego obiektu na podstawie konstruktora.

var osoba1 = new klient("Kowalski","Jan","kierowca");

8. Właściwość prototype

Prototype służy do dodawania metod i właściwości do wszystkich obiektów danej klasy.

function klient() {
this.imie = "Paweł";
this.nazwisko = "Bielski";
}
klient.prototype.zawod = "kierowca";
klient.prototype.pisz = function() {
document.write(this.imie + " " + this.nazwisko);
}

Każdy obiekt utworzony z konstruktora klient będzie posiadał właściwości i metody dodane przez prototype.
